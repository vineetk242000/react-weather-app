## Requirements
```
nodejs
npm
```
## Setup
First Clone this repository 

```
$ git clone https://github.com/vineetk242000/react-weather-app.git
```
Install dependencies
```
npm install # or yarn
```
Run it
```
npm start # or yarn start
```

## API Refrence
```
https://openweathermap.org/api
```
